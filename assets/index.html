Create, build and deploy your own simple, static and responsive microsite.

download

Tools

Easy, automated workflow.
Minimal, responsive CSS thanks to skeleton.
Deploy your site to Github. The best part? It's free!
Browsersync let's you view your site on multiple browsers.

Prerequisites

Install node, join github, set up git and create your site repository

Instructions

Copy microcosm.zip contents into your username.github.io directory
Your folder should look as follows
folder/
└── username.github.io/
    ├── assets/
    │   ├── img/
    │   │   ├── GitHub-Mark-Light.png
    │   │   ├── blurred_lines-min.jpeg
    │   │   └── favicon.png
    │   ├── index.html
    │   ├── js/
    │   │   └── functions.js
    │   └── styles/
    │       ├── microcosm.scss
    │       ├── normalize.css
    │       └── skeleton.css
    └── tasks/
    ├── gulpfile.js
    ├── package.json
Install the necessary components and start your server with these commands
$ npm install
$ gulp serve

Design

Make all of your changes in the assets directory
Maintain the established file structure or update gulp tasks to handle any changes

Deploy

Execute these commands to update your repository and automatically deploy your site
$ git add . -u
$ git commit -m 'My Microsite'
$ git push
